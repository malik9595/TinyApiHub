Guide

Below you'll find examples using Fetch API but you can TinyApiHub with any other language. You can copy paste the code in your browser console to quickly test TinyApiHub.

Getting a resource

Note: replace https://tinyapihub.com with actual home domain as you see at the top of your browser window.

fetch ('https://tinyapihub.com/users/1' )
.then (response =>response. json())
.then (json => console. log(json))
👇 Output
{
  _id: 6614fcabf15055607c4d38bd,
  name: '...',
  email: '...',
    /* ... */
  add: '...',
}

fetch ('https://tinyapihub.com/users' )

Creating a resource

fetch('https://tinyapihub.com/todolist', {
  method:'POST' ,
  body: JSON .stringify({
    task : 'Buy groceries',
    completed : false,
    time : '4:00 PM',
    id : 1,
   }),
  headers: {
     'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));
👇 Output
{
 _id: 6614fcdbf15055607c4d3948
 id: 1
 task: Buy groceries
 completed: false
 time: 4:00 PM
}

Important: resource will not be really updated on the server but it will be faked as if.

Updating a resource

fetch('https://tinyapihub.com/posts?id=1', {
  method:'PUT' ,
  body: JSON .stringify({
    username : 'wildlife_watcher',
    title : 'Rare Sangai Deer',
    body : 'Witnessing the graceful Sangai deer',
    userId : 1,
   }),
  headers: {
     'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));
👇 Output
{
 _id: 6614fcd8f15055607c4d3920
 id: 1
 username: wildlife_watcher
 title: Rare Sangai Deer
 body: Witnessing the graceful Sangai deer
}

Important: resource will not be really updated on the server but it will be faked as if.

Patching a resource

fetch('https://tinyapihub.com/todolist?id=3', {
  method:'PATCH' ,
  body: JSON .stringify({
    task : 'Update resume',
  headers: {
     'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));
👇 Output
{
 _id: 6614fcdbf15055607c4d3957
 id: 3
 task: update resume
 body: '...'
 completed: '...'
 time: '...'
}

Important: resource will not be really updated on the server but it will be faked as if.

Deleting a resource

fetch('https://tinyapihub.com/posts', {
  method:'DELETE'
}

Important: resource will not be really updated on the server but it will be faked as if.

Filtering resources

Basic filtering is supported through query parameters.

// This will return all the posts that belong to the first user
fetch ('https://tinyapihub.com/posts?id=1' )
.then (response =>response. json())
.then (json => console. log(json))
