Free fake and reliable API for testing and prototyping.

If you're new to API handelling, this collection is a great place to start exploring APIs that are free to test, specifically updated for 2024.

Run this code here, in a console or from any site:

fetch ('https://tinyapihub.com/todos/1' )

When to use

TinyApiHub is a free online REST API that you can use whenever you need some fake data. It can be in a README on GitHub or simply to test things locally.

Resources

FreeFakeAPI comes with a set of 3 common resources:

/users 50 users
/posts 20 posts
/todolist 20 todolist

Note: see guide for usage examples.

Routes

All HTTP methods are supported. You can use http or https for your requests.

GET /users
GET /users/1
GET /posts/2
GET /todolist/?id=3
POST /users
PUST /users/?id=1
PATCH /users/?id=1
DELETE /users/?id=1

Note: see guide for usage examples.
